### Project created to explore capabilieties of new features of next.js 13

### Prequisites

- node v16

### Launching project

- npm install
- npm run dev
- npx prisma db push
- npx prisma db seed

### New features explored:

- appDir
- using Server-side components instead of getStaticProps
- New way for dynamic routing
